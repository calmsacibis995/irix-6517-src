System Information > Introduction
 

 

The System Information section of SGI Embedded Support Partner User Interface provides access to the data collected by various ESP daemons. This data is grouped into various reports. All available reports provide an option to walk across archived data, if available. A subset of available data can be requested by giving appropriate time range. The available reports include:

  • Hardware Report lets you browse various hardware components that are available on your system (and subscribed systems if applicable). The data is presented in a drill down fashion.
  • Software Report shows the current software components installed on your system (and if applicable, subscribed systems).
  • System Changes Report can provide you with a historical view of transformations that the system went through (with regards to Hardware, Software and System Information) in the past.
  • The Events Registered Report contains a rich set of information about events that happened on your system and are captured by SGI Embedded Support Partner.
  • If you have configured SGI Embedded Support Partner to take proactive actions for any event, you can view the actions taken by ESP using Actions Taken Report.
  • The Availability Report provides a statistical view of your system interrupts along with the specific time and date when interrupt happened.
