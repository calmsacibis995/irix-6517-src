Setting Up a System

Hardware and Software Requirements

The system must meet the requirements for running as a partitioned system. This includes having an Origin 2000 with at least 2.3 hubs and having that system partitioned into one of the supported configurations. The system has to be running at least version 6.5.3 of Irix and have all the necessary clshm files installed and modules configured into the operating system.

Configuring the Operating System

/var/sysgen/system/irix.sm must have the following modules changed from being EXCLUDE to INCLUDE on all partitions in the system:

  • partition
  • xpc
  • cl
  • clshm
  • if_cl (optional)

And once this is changed all systems should be "autoconfig"ed.

To partition down the system see the mkpart command man pages on the system.

Necessary Files

The following files need to have been installed on your system:

  • /usr/etc/clshmctl
  • /usr/etc/clshmd
  • /usr/etc/clshm_diag
  • /etc/config/clshmd
  • /etc/init.d/sn0start
  • /usr/include/xp_shm.h
  • /usr/lib32/libclshm.so
  • /usr/lib64/libclshm.so

Starting Up the Daemon

Once the system is up and running as a partitioned system, the clshm daemon must be turned on. This is done using the /usr/etc/clshmctl command. Or if you want the daemon to be started up every time the system boots up then do a "chkconfig clshmd on" and the next time the system boots up the daemon will be on. To make sure that all partitions are up and have their daemons running run the clshm_diag command.
