Banyan 6.2 Desktop Performance Subsystems



The installed subsystems for 6.2 timings are kept relatively constant from one release to the next. Some subsystems such as new "books" are added from one version to another. To keep track of changes such as this, I will be appending a "versions" list to each set of timings.

For every release, the EXACT SAME subsystems are installed for both the EFS and XFS timings. This includes books and icons in the icon panels. The same testing routine is followed as well, so both EFS and XFS have a growing log file in the home directory.

If programs run slower under XFS than EFS it most likely is not due to "filesystem contents".
